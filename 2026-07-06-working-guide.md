# CSI Indaba Day 2 Working Guide

Date: 2026-07-06
*Last updated: Jul 06 2026*

## What We Have Been Doing

We have been building the Day 2 programme in three repeatable steps:

1. Check the programme card content.
2. Add or update the speaker details and speaker links.
3. Build or fix the click-through session page so it matches the card.

## Step 1: Check The Programme Card

We first confirm the card text before changing anything else.

What we check:

- Time
- Room
- Session title
- Session type
- Main description
- Speaker names
- Speaker organisations
- Speaker thumbnails

Example work done:

- `Round Room` was checked first before the speaker cards were built.
- `Podcast Room` was checked next before the click page was updated.

## Step 2: Add Or Fix Speaker Details

Once the card content is confirmed, we update the speaker details.

What we do:

- Add the correct image link
- Add the speaker name
- Add the organisation or role
- Add a short bio when available
- Use placeholder content when the final bio is not ready yet

Example work done:

- `Alef Meulenberg` got an image and bio.
- `Nicole Solomon` got an image and bio.
- `Yumna Toefy` got a placeholder-style bio and image.
- `Sello Hatang`, `Fikile Kuhlase`, and `Princess Gabo` were added for the Podcast Room session.

## Step 3: Build The Click-Through Session Page

After the card is right, we make the click page match it.

What we add on the session page:

- Hero image
- Room label
- Time
- Session title
- Main description
- Speaker block or speaker cards
- `View profile` links
- Back links to the programme

Example work done:

- The Podcast Room session page was created at `session-pages/building-accountability-csi.html`.
- The page was updated to match the programme card copy.
- The speaker section was changed to three separate cards so it feels like the Round Room layout.

## The Pattern We Kept Repeating

For each new card, we followed the same order:

1. Confirm the card content.
2. Add the speaker images and names.
3. Create or update the session page click-through.
4. Add profile links for each speaker.
5. Match the look and feel of the existing main-room cards.

## Current State

What is now in place:

- Round Room main programme card
- Podcast Room main programme card
- Podcast Room click-through session page
- Speaker profile links for the session speakers
- Placeholder profile support for Princess Gabo

## Good Rule Going Forward

When adding the next Day 2 card, keep the same rhythm:

- main programme card first
- speaker cards second
- click-through page third

That keeps the site consistent and avoids missing links or mismatched content.
